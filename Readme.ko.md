# ESP32 LittleFS 통합 관리 도구 (LittleFS Toolset)

이 프로젝트는 ESP32 장치에서 사용하는 **LittleFS 파일 시스템을 GUI 환경에서 직관적으로 관리**하기 위한 통합 도구 모음입니다. 터미널 명령어나 복잡한 인자 입력 없이 시각적인 인터페이스를 통해 파티션 분석부터 파일 편집, 기기 업로드까지 전 과정을 제어할 수 있습니다.

## 핵심 도구: LittleFS Explorer (`littlefs_explorer.py`)

`littlefs_explorer`는 프로젝트의 메인 인터페이스로, 흩어져 있는 개별 기능들을 하나의 창에서 사용할 수 있게 통합한 애플리케이션입니다.

### 주요 기능
*   **파티션 테이블 자동 분석**: ESP32 장치의 `0x8000` 오프셋을 읽어 파티션 구성을 시각적인 블록 형태로 표시합니다. 사용자는 LittleFS 파티션을 한눈에 찾고 선택할 수 있습니다.
*   **기기 직접 통신 (Read/Write)**: 선택한 포트와 파티션을 대상으로 즉시 데이터를 가져오거나(Dump), 수정한 내용을 다시 기기에 기록(Flash)합니다.
*   **실시간 파일 관리 (CRUD)**:
    *   LittleFS 바이너리 내부의 파일 목록을 트리 구조로 표시합니다.
    *   파일 더블 클릭을 통한 텍스트 편집 기능을 제공합니다.
    *   우클릭 메뉴를 통해 파일/폴더 생성 및 삭제가 가능합니다.
*   **드래그 앤 드롭 지원**: 로컬 PC의 파일들을 탐색기 창으로 끌어다 놓는 것만으로 간편하게 파일 시스템에 추가할 수 있습니다.
*   **스토리지 모니터링**: 현재 파일 시스템의 실제 용량과 파티션 크기를 실시간으로 비교하여 표시하며, 용량 초과 시 시각적인 경고를 제공합니다.

## 설치 및 실행

### 요구 사항
*   **Python 3.x**
*   **의존성 패키지**: `esptool`, `littlefs-python`, `pyside6` (PySide6는 GUI 실행에 필수입니다.)

### 빠른 시작 (Windows)
1.  `venv_install.ps1` 스크립트를 실행하여 가상 환경 및 필요한 라이브러리를 자동 설치합니다.
2.  설치 완료 후 다음 명령어로 실행합니다:
    ```bash
    .\venv\Scripts\python.exe littlefs_explorer.py
    ```

## 프로젝트 구성

*   **GUI 에디터**: `littlefs_explorer.py` (메인 실행 파일)
*   **백엔드 스크립트**: LittleFS 조립/해제, 기기 덤프/플래싱을 담당하는 개별 모듈들.
*   **테스트 코드**: ESP32 기기에서 LittleFS 동작을 검증하기 위한 `src/main.cpp` (Arduino 환경용).
*   **이미지 자원**: GUI 버튼에 사용되는 아이콘들 (`img/` 폴더).

## 참고 문서

*   각 개별 스크립트(`mklittlefs.py`, `dump_littlefs.py`, `flash_littlefs.py` 등)의 상세한 기능 명세와 CLI 사용법은 **Scripts.ko.md** 파일에서 확인하실 수 있습니다.
